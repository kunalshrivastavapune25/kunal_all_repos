# Building an AI Agent - Step by Step

## Step 1: Define Purpose
- Problem it solves
- Target user
- Success criteria

## Step 2: Choose Tools
- Search
- Database
- Email
- Calculator
Each tool should do ONE job well.

## Step 3: Design System Prompt
- Define role
- Set tone
- Add constraints
- Provide context

## Step 4: Memory
- Conversation history
- User preferences
- Task progress

## Step 5: Execution Loop
1. Receive input
2. Decide tool
3. Execute tool
4. Process result
5. Respond
6. Repeat until done

## Step 6: Error Handling
- API failure
- Invalid input
- Clarification
- Fallback

## Step 7: Testing
- Happy path
- Edge cases
- Improve prompts
