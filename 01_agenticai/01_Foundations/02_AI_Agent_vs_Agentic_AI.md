# AI Agent vs Agentic AI

| Aspect | AI Agent | Agentic AI |
|--------|----------|------------|
| Core Idea | Follows instructions | Achieves goals autonomously |
| Autonomy | Low | High |
| Goal Handling | Fixed goal | Breaks into sub-goals |
| Memory | Short-term | Short + Long term |
| Tool Usage | Uses given tools | Selects tools independently |
| Workflow | Linear | Iterative and dynamic |
| Example | FAQ Bot | Autonomous research agent |

---

# Traditional vs Generative vs Agentic AI

| Aspect | Traditional | Generative | Agentic |
|--------|------------|------------|----------|
| Core Idea | Rules & prediction | Content generation | Goal execution |
| Human Involvement | High | Medium | Low |
| Tool Usage | Minimal | Prompted | Self-selected |
| Example | Spam Filter | ChatGPT | AutoGPT / LangGraph |
